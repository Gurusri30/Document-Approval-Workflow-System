This project is a simple Document Management System developed using Java Servlets, JSP, and JDBC following the MVC architecture. It allows users to add, view, update, and delete documents stored in a database. The MainServlet handles requests, the DAO manages database operations, and JSP pages display the data. The application runs on Apache Tomcat and demonstrates basic Java web development concepts.


Features

 Add New Document

 View All Documents

 Update Document

 Delete Document

 View Document Details

Database connectivity using JDBC

MVC Architecture (Servlet + JSP + DAO)
